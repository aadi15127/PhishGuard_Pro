# PhishGuard Pro 🛡️

A cyberpunk-themed **Cybersecurity Awareness Hub** designed to train users on identifying phishing attempts. 

## Features
- **Radar Dashboard**: Real-time visual threat simulation.
- **Global Leaderboard**: Sync scores across the web using JSONBin API.
- **Link Inspector**: A heuristic scanner for common phishing red flags.
- **Tactical Quiz**: Interactive learning module with immediate feedback.

## Tech Stack
- HTML5 / CSS3 (Custom Grid System)
- Vanilla JavaScript
- JSONBin.io (No-SQL Cloud Storage)

## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
3. *Note:* Ensure you have your own API keys for JSONBin if you wish to host your own leaderboard.

## Security Note
This is an educational tool. The "Link Inspector" is for training purposes and does not replace actual antivirus software.
